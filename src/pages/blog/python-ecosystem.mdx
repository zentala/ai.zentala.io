## Rozpoczęcie Pracy z Pythonem

### Wirtualne Środowiska
- **venv**: Narzędzie do tworzenia izolowanych środowisk Pythona. Zapobiega problemom z zależnościami między projektami.
  [Dokumentacja venv](https://docs.python.org/3/library/venv.html)
- **pipenv**: Automatyzuje zarządzanie pakietami i wirtualnymi środowiskami. Łączy Pipfile, pip i venv w jednym narzędziu.
  [Strona pipenv](https://pipenv.pypa.io/en/latest/)

### Frameworki Webowe
- **Django**: Wysokopoziomowy framework do szybkiego tworzenia bezpiecznych i skalowalnych aplikacji webowych.
  [Strona Django](https://www.djangoproject.com/)
- **Flask**: Lekki i elastyczny mikroframework do tworzenia aplikacji webowych.
  [Strona Flask](https://flask.palletsprojects.com/)

### Biblioteki do Analizy Danych
- **Pandas**: Potężna biblioteka do analizy i manipulacji danymi.
  [Strona Pandas](https://pandas.pydata.org/)
- **NumPy**: Podstawowa biblioteka do obliczeń naukowych w Pythonie.
  [Strona NumPy](https://numpy.org/)
- **SciPy**: Biblioteka do matematycznych algorytmów i funkcji.
  [Strona SciPy](https://www.scipy.org/)
- **Matplotlib**: Biblioteka do tworzenia statycznych, interaktywnych i animowanych wizualizacji w Pythonie.
  [Strona Matplotlib](https://matplotlib.org/)
- **Seaborn**: Biblioteka do tworzenia atrakcyjnych i informacyjnych wykresów statystycznych.
  [Strona Seaborn](https://seaborn.pydata.org/)

### Machine Learning i AI
- **TensorFlow**: Biblioteka do uczenia maszynowego i głębokiego uczenia od Google.
  [Strona TensorFlow](https://www.tensorflow.org/)
- **PyTorch**: Otwarta biblioteka uczenia maszynowego od Facebooka.
  [Strona PyTorch](https://pytorch.org/)
- **scikit-learn**: Prosta i efektywna biblioteka do uczenia maszynowego.
  [Strona scikit-learn](https://scikit-learn.org/)
- **Keras**: Interfejs do sieci neuronowych, działający na TensorFlow.
  [Strona Keras](https://keras.io/)

### Web Scraping
- **Beautiful Soup**: Biblioteka do parsowania dokumentów HTML i XML.
  [Strona Beautiful Soup](https://www.crummy.com/software/BeautifulSoup/)
- **Scrapy**: Ramka do ekstrakcji danych z witryn internetowych.
  [Strona Scrapy](https://scrapy.org/)

### Asynchroniczność i Równoległość
- **Asyncio**: Biblioteka do pisania kodu równoległego za pomocą async/await.
  [Dokumentacja asyncio](https://docs.python.org/3/library/asyncio.html)
- **aiohttp**: Asynchroniczny klient HTTP i serwer dla asyncio.
  [Strona aiohttp](https://docs.aiohttp.org/en/stable/)

### Testowanie
- **pytest**: Elastyczne narzędzie do testowania.
  [Strona pytest](https://pytest.org/)
- **unittest**: Wbudowany framework do testowania jednostkowego.
  [Dokumentacja unittest](https://docs.python.org/3/library/unittest.html)

### ORM (Object-Relational Mapping)
- **SQLAlchemy**: Biblioteka ORM do interakcji z bazami danych.
  [Strona SQLAlchemy](https://www.sqlalchemy.org/)
- **Django ORM**: Wbudowany ORM w Django do modelowania danych.
  [Dokumentacja Django ORM](https://docs.djangoproject.com/en/3.2/topics/db/models/)

### Deployment i Zarządzanie Aplikacjami
- **Docker**: Narzędzie do tworzenia, wdrażania i uruchamiania aplikacji w kontenerach.
  [Strona Docker](https://www.docker.com/)
- **Kubernetes**: System do automatyzacji wdrażania, skalowania i zarządzania aplikacjami kontenerowymi.
  [Strona Kubernetes](https://kubernetes.io/)

### GUI i Integracje
- **Tkinter**: Standardowy interfejs GUI dla Pythona.
  [Dokumentacja Tkinter](https://docs.python.org/3/library/tkinter.html)
- **PyQt**: Zestaw narzędzi do tworzenia GUI, działający na Qt.
  [Strona PyQt](https://riverbankcomputing.com/software/pyqt/intro)

### Microframeworki i API
- **FastAPI**: Nowoczesny, szybki framework do budowy API.
  [Strona FastAPI](https://fastapi.tiangolo.com/)
- **Falcon**: Framework do tworzenia API.
  [Strona Falcon](https://falconframework.org/)
